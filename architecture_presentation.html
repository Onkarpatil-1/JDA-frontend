3. Data Persistence

A "Tri-Modal" storage strategy ensures high-speed dashboards, reliable records, and complex AI memory.

PostgreSQL 16

Purpose: Structured Application Records

Uses JSONB for flexible application attributes and strict relational schemas for audit logs and SLA status.

ChromaDB / Pinecone

Purpose: Vector Store for RAG

Stores embeddings of JDA Rules, Handbooks, and past MoM decisions to power the AI Officer assistant.

Redis Stack

Purpose: Real-time Cache

Caches the "Model Code of Conduct" global states and active dashboard KPIs for sub-100ms load times.
